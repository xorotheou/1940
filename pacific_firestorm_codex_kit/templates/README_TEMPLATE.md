# Pacific Firestorm

Original Godot 4 vertical-scrolling aerial shooter with arcade gameplay and ultra-realistic Pacific air-combat visuals.

## Status
Prototype planning kit. Implement phases in order.

## Engine
Godot 4.x, GDScript.

## Core Rule
Arcade gameplay. Realistic visuals. Original identity.

## How to Open
1. Install Godot 4.x.
2. Clone or open this repository.
3. Open `project.godot`.
4. Set `scenes/main/Main.tscn` as the main scene.
5. Press Play.

## Controls
- W/A/S/D or arrow keys: move
- Space: fire
- Ctrl: bomb
- Shift: evade
- Esc: pause
